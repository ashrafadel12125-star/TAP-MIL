# 📧 TempMail Pro - بريد إلكتروني مؤقت عصري

موقع متقدم لتوليد بريد إلكتروني مؤقت بتصميم عصري وحديث، يعمل بشكل حقيقي تماماً ويدعم استقبال الرسائل الفعلية.

## ✨ الميزات الرئيسية

- 🎯 **توليد بريد مؤقت تلقائي** - إنشاء عنوان بريد في ثانية واحدة
- 📨 **استقبال الرسائل الفعلية** - استقبل رسائل التفعيل والتحقق بشكل حقيقي
- 🔄 **تحديث تلقائي** - تحديث الرسائل كل 3 ثوان تلقائياً
- 📋 **إدارة كاملة** - نسخ، تحديث، تغيير، أو حذف البريد بسهولة
- ⏱️ **عداد الصلاحية** - عرض الوقت المتبقي قبل انتهاء البريد
- 🎨 **تصميم عصري جذاب** - واجهة احترافية مع تدرجات لونية
- 📱 **متوافق مع الهواتف** - يعمل بشكل مثالي على جميع الأجهزة
- 💰 **مدعوم بالإعلانات** - مساحات محسّنة لإعلانات Google AdSense

## 🚀 البدء السريع

### المتطلبات
- متصفح ويب حديث
- اتصال إنترنت

### الاستخدام
1. افتح الموقع في المتصفح
2. سيتم إنشاء بريد مؤقت تلقائياً
3. استخدم الأزرار للتحكم بالبريد
4. انتظر استقبال الرسائل

## 🔧 الأزرار الرئيسية

| الزر | الوظيفة |
|-----|--------|
| 📋 نسخ | نسخ عنوان البريد إلى الحافظة |
| 🔄 توليد بريد جديد | إنشاء بريد جديد تماماً |
| 🔁 تغيير البريد | استبدال البريد الحالي |
| 🗑️ حذف البريد | حذف البريد والرسائل |
| ⏱️ تحديث تلقائي | تفعيل/إيقاف التحديث الدوري |

## 🔌 تقنيات مستخدمة

### Frontend
- **HTML5** - الهيكل والمحتوى
- **CSS3** - تصميم عصري مع تدرجات وانتقالات سلسة
- **Bootstrap 5** - نظام الشبكات والمكونات
- **JavaScript ES6+** - المنطق والتفاعلات

### API المستخدمة
- **Mail.gw API** - خدمة توليد البريد المؤقت
  - `GET /domains` - الحصول على قائمة النطاقات المتاحة
  - `POST /accounts` - إنشاء حساب بريد جديد
  - `GET /messages` - جلب الرسائل الواردة
  - `GET /messages/{id}` - تفاصيل رسالة محددة
  - `DELETE /accounts/{id}` - حذف حساب بريد
