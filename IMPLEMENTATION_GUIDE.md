## 🚀 الميزات الرئيسية

### 1️⃣ الصفحة الرئيسية (index.html)
- **التنقل العلوي**: روابط سهلة للمقالات والخطط المدفوعة ولوحة التحكم
- **إنشاء البريد التلقائي**: توليد بريد مؤقت فوري عند التحميل
- **الإدارة الكاملة**: نسخ، تحديث، تغيير، حذف البريد
- **عرض الرسائل**: استقبال وعرض الرسائل الفعلية في الوقت الفعلي
- **قسم المقالات**: عرض المقالات المنشورة من لوحة التحكم
- **الإعلانات**: مساحات محسّنة لـ Google AdSense

### 2️⃣ لوحة التحكم (admin/dashboard.html)
**تسجيل الدخول التلقائي:**
```
البريد: admin@tempmail.pro
كلمة المرور: TempMail@2026
```

**الأقسام المتاحة:**
1. **لوحة البيانات**: إحصائيات عامة
2. **المقالات**: كتابة ونشر مقالات جديدة
3. **الصفحات**: إدارة صفحات إضافية
4. **الأقسام**: إضافة أقسام جديدة للموقع
5. **الإعدادات**: تعديل إعدادات الموقع و Google AdSense

### 3️⃣ صفحة الخطط المدفوعة (premium.html)
**ثلاث خطط اشتراك:**

| الميزة | الأساسي | احترافي | مؤسسي |
|--------|---------|-----------|--------|
| السعر | $4.99 | $9.99 | $19.99 |
| البريد يومياً | 50 | 200 | غير محدود |
| التخزين | 7 أيام | 30 يوم | 90 يوم |
| الدعم | لا | نعم | 24/7 |
| API | لا | لا | نعم |

**دمج Stripe:**
- دفع آمن عبر بطاقات ائتمان
- جاهز للتشغيل مع Stripe API
- معالجة فوية الأخطاء

## 📖 كيفية الاستخدام

### 1. تشغيل الموقع محلياً
```bash
# افتح الملف مباشرة في المتصفح
file:///path/to/index.html
```

### 2. استخدام الصفحة الرئيسية
- سيتم توليد بريد تلقائياً عند التحميل
- استخدم الأزرار للتحكم بالبريد
- الرسائل تحدث تلقائياً كل 3 ثواني

### 3. الوصول إلى لوحة التحكم
- انقر على أيقونة القفل <i class="fas fa-lock"></i> في الرأس
- أو انتقل مباشرة إلى `admin/dashboard.html`
- سجل دخول باستخدام البيانات الافتراضية

### 4. نشر مقالة
```javascript
// في لوحة التحكم:
1. اذهب إلى "المقالات"
2. اكتب العنوان والملخص والمحتوى
3. انقر "حفظ المقالة"
4. ستظهر تلقائياً في الصفحة الرئيسية
```
